## 硬件:

+ 相机: 大恒相机 (`MER-131-210U3C`)
+ 计算: Jetson Xavier NX
+ 通讯: USB TO TTL

## 软件

+ CUDA (`11.4`)
+ OpenCV (`4.7.0 with CUDA`)
+ DahengSDK (`1.2.2206.9161 2022-06-16`)
